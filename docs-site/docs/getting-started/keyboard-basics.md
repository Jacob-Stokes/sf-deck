# Keyboard basics

The dozen keys you'll use 90% of the time. Press `?` on any screen
for the full list.

## Movement

| Key | What |
|---|---|
| `↑` `↓` `j` `k` | Move cursor up / down one row |
| `Ctrl+↑` `Ctrl+↓` | Jump ~5 rows |
| `PgUp` `PgDn` `Ctrl+U` `Ctrl+D` | Page up / down |
| `g` / `G` | Go to top / bottom |
| `Enter` | Drill in / activate |
| `Esc` | Back / cancel |

## Tabs and subtabs

| Key | What |
|---|---|
| `1` – `9` | Jump to tab N |
| `0` | Open the "more tabs" modal |
| `Shift+1` – `Shift+9` (or `! @ # $ % ^ & * (`) | Jump to subtab N |
| `Tab` / `Shift+Tab` | Next / previous subtab |

## Chips (filter views)

Every list-shaped tab has a chip strip — saved filters you can flip
between.

| Key | What |
|---|---|
| `[` `]` | Cycle chips left / right |
| `Shift+←` `Shift+→` | Same |
| `V` | Open the chip manager |
| `F` | Toggle the cursored chip as favourite |

## Search and filter

| Key | What |
|---|---|
| `/` | Filter the current list — type to narrow |
| `Ctrl+F` | Global search across metadata + records |

## Org switching

| Key | What |
|---|---|
| `'` | Focus the org rail |
| then `j` / `k` | Move between orgs |
| then `Enter` | Switch to the highlighted org |
| then `space` | Fold / unfold an org group |
| then `a` | Open the add-org picker |

Comparing metadata across orgs is the `/compare` tab, not the org rail.

## Panes

| Key | What |
|---|---|
| `\` | Toggle the right sidebar |
| `Ctrl+\` | Toggle stacked vs side-by-side sidebar |
| `\|` | Toggle the left rail |
| `i` | Open the cursored row's sidebar info in a modal |

## Working with items

| Key | What |
|---|---|
| `K` | Quick-collect cursored item into the loaded [dev project](../concepts/dev-projects.md) (toggle) |
| `Ctrl+K` | Collect cursored item — pick which project |
| `t` | Open the tag picker for the cursored item |
| `T` | Tag every visible row (bulk) |
| `o` | Open the cursored item in Lightning (in your browser) |
| `Ctrl+O` | Open menu — pick a target |
| `y` | Yank (copy to clipboard) — id / URL / etc. |

## SOQL

On `/soql` only:

| Key | What |
|---|---|
| `i` | Enter the editor (input mode) |
| `Enter` (from input mode) | Run the query |
| `Escape` | Leave input mode |
| `s` | Save the current query to the library |
| `T` | Toggle Tooling API |

## When all else fails

| Key | What |
|---|---|
| `?` | Show every key the current screen understands |
| `Ctrl+C` | Quit sf-deck |
| `Ctrl+R` | Refresh all loaded data for the active org |
