# sf-deck surface inventory

Auto-generated reference of every Tab / Subtab / drill destination,
plus the hooks each surface has wired (List / Open / Chips / Activate).
Generated by `go run ./cmd/inventory`. Hand-edit human descriptions
after running ‚Äî re-running OVERWRITES the file.

## How to read this

- **List** ‚Äî has a tabulated row list (cursor + search)
- **Open** ‚Äî `o` / `ctrl+o` open the Salesforce Lightning destination
- **Chips** ‚Äî has a chip strip (filter / view-switching)
- **Drill** ‚Äî `enter` drills into a detail tab; arrow shows destination
- **Sidebar** ‚Äî has a right-pane context panel
