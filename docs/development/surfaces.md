# sf-deck surface inventory

Auto-generated reference of every Tab / Subtab / drill destination,
plus the hooks each surface has wired (List / Open / Chips / Activate).
Generated by `go run ./cmd/inventory`. Hand-edit human descriptions
after running — re-running OVERWRITES the file.

## How to read this

- **List** — has a tabulated row list (cursor + search)
- **Open** — `o` / `ctrl+o` open the Salesforce Lightning destination
- **Chips** — has a chip strip (filter / view-switching)
- **Drill** — `enter` drills into a detail tab; arrow shows destination
- **Sidebar** — has a right-pane context panel
